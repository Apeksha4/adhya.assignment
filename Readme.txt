# Web Development Assignment - Product Catalog + Shopify Section


## Task 1: PHP + MySQL Product Catalog

1. Import product_catalog.sql in phpMyAdmin.

2. Place index.php in your local server's htdocs folder (XAMPP/WAMP)
.
3. Open http://localhost/index.php in browser.

4. You will see:
  
 - Product list with dummy data.
 
  - Category filter dropdown.
  
 - Add product form (with validation).


## Task 2: Shopify/HTML Mockup

1. Open shopify.html in any browser.

2. You will see a featured product card (image, name, price, Buy Now button).

3. Hover effect and responsive layout are included
.
